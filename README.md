Smart Clinic Management System

Project Overview

The Smart Clinic Management System is a web-based healthcare application developed using PHP and MySQL. It enables administrators and patients to manage appointments, patient records, and medical records efficiently.

Features

- User Login and Authentication
- Patient Management
- Appointment Booking and Scheduling
- Medical Records Management
- Search Functionality
- Edit and Delete Operations
- Admin Dashboard
- Patient Dashboard

Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP
- Visual Studio Code

Database Tables

- Users
- Patients
- Appointments
- Medical Records

Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Create the database in phpMyAdmin.
4. Import the SQL database.
5. Copy the project folder into the "htdocs" directory.
6. Open:

http://localhost/clinic_system/
